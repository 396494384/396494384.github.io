- ##### 创建git仓库   
  项目初始化的流程
~~~
  //初始化本地仓库，在当前目录下生成 .git 文件夹
  git init
  //默认在当前目录下创建和版本库名相同的文件夹并下载版本到该文件夹下
  git clone <远程仓库的网址> 
~~~

- ##### 代码的拉取  
  这里只是基础的代码拉取
~~~
git pull origin <远程仓库的分支>
~~~
